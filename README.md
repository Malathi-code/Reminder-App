# Reminder App

## 📌 Description

This is a Python-based desktop application that allows users to set reminders and receive notifications.

## 🚀 Features

* Add reminders with time (AM/PM)
* Desktop notifications
* Sound alert
* Snooze option
* Save reminders (JSON)
* Delete and clear reminders

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* Plyer (Notifications)
* JSON

## ▶️ How to Run

1. Download the project
2. Run:
   python reminder.py

## 🎯 Purpose

This app helps users manage tasks and get notified at the correct time.
